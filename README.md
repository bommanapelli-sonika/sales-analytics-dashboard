# 📊 Sales Analytics Dashboard

🔗 Live Demo: https://bommanapelli-sonika.github.io/sales-analytics-dashboard/

---

## 🚀 Project Overview
This project is a Sales Analytics Dashboard built using HTML, JavaScript, and Chart.js.  
It fetches real-time data from an API and visualizes it using interactive charts.

---

## ✨ Features
- 📊 Bar Chart for data visualization  
- 🥧 Pie Chart for category analysis  
- 🔄 Real-time data using API  
- 📈 Basic analysis (Total, Highest, Lowest)  

---

## 🛠️ Technologies Used
- HTML  
- JavaScript  
- Chart.js  
- API (Fetch)

---

## ▶️ How to Run
1. Download the project  
2. Open `index.html` in browser  
3. Ensure internet connection  

---

## 💼 Use Case
This project demonstrates data visualization and frontend development skills, useful for analytics dashboards.
